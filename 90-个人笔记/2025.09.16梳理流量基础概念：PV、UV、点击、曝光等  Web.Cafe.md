# \[2025.09.16\]梳理流量基础概念：PV、UV、点击、曝光等

大家好，中午开个简短的#哥飞小课堂 。

给大家梳理一些基础概念，我发现总有朋友会搞错。

Google Search Console（GSC）、Bing Webmaster Tools（BWT） 里的数据，是我们的网站在这些搜索引擎里的表现数据。

当有人在搜索引擎里搜索了某个关键词，如果搜索结果里出现了我们的网页，那么曝光（展示）数据就会+1，如果这个人还点击了结果里的我们网页，那么点击数据就会+1。

与此同时，我们网站的 UV 就+1 。 假如这个人访问了我们3 个网页，则我们网站的 PV+3。 UV 和 PV，在 GSC 和 BWT 里是看不到的。

千万不能把上面的曝光当做是 PV。

假如我们的网站放了 Adsense 广告代码。

一个页面假设出现了 3 个广告位，而这个人打开 3 个页面时，把页面上的 3 个广告位都看到了，那么 Adsense后台的广告展示次数+9 （3 页x 3广告）。

如果这个人看到我们网站的广告，感兴趣了，点击了，打开了广告主的网站。

那么此时 Adsense 后台的广告点击数据就会+1。

由于 GSC 和 BWT 里有展示（曝光）和点击，而 Adsense 里也有展示和点击，于是有的群友认为 GSC 和 BWT 里的展示应该等于Adsense里的展示，点击也应该相等。

这就错得离谱了，不能因为名字一样，就认为是同一个数据啊。

与 GSC 和 BWT 里的曝光和点击更相近的概念是 Ads 广告后台的数据。

我们花钱在谷歌投放广告，我们的广告被展示了一次，曝光数+1，有人感兴趣点击了我们广告，打开了我们网站，此时广告的点击数据+1，与此同时我们网站的 UV+1。

不一样的是，GSC 和 BWT 里的曝光和点击都是免费的，我们不需要花钱。而 Ads 里的曝光和点击我么需要花钱买。

越是基础概念，大概越是要去搞清楚，不然就会出现认知错误。

不要想当然，不要迷糊，要清晰。

好了，今天的简短小课堂就上完了。

对了，GSC 和 BWT 里的曝光，理论上来说，是需要用户翻页到了，看到你的结果里，才算一个的。

举例，你排在第三页，用户只看第一页，此时不算曝光，只有用户真的翻到了第三页，曝光才+1。

不过呢，有些爬虫，为了抓取更多结果，会翻页，以及之前谷歌提供一次性看100 条结果的参数，于是即使排到了后几页，也是有可能在 GSC 里显示有曝光。

还有种情况是你的网页在 A 地区排第三页，但是在 B 地区是排第一页的，B 地区用户搜索后在第一页就看到你的网页了，此时曝光也会+1 。

但GSC 显示的排名数据是平均排名，我们会看到我们的网页平均排名肯定是不在第1页的，于是就会有困惑，到底曝光哪来的？
