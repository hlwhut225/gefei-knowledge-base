# 为什么谷歌会给新站机会？

如果大家看了我上上周的公益直播视频，就知道，我反复说了一个点，谷歌为了保住自己搜索巨头地位，害怕失去最新鲜的网站供应，所以建立了最强的爬虫系统，不惜花费大量金钱也要去爬取尽可能全的网页。

谷歌最怕的是爬不全，而不是爬到了垃圾页面浪费资源。

只要爬下来了，谷歌就有很多办法来识别一个网页的质量，其中就包括刚才大家都回答到了的，给新站一个机会，试试新站的到底能不能打。 ![image.png](%E4%B8%BA%E4%BB%80%E4%B9%88%E8%B0%B7%E6%AD%8C%E4%BC%9A%E7%BB%99%E6%96%B0%E7%AB%99%E6%9C%BA%E4%BC%9A%EF%BC%9F%20%20Web.Cafe/2c7719c6294e489496766c9286965bd6.png) 从刚才的截图可以看到，有很多网站都差不多是这样，有曝光了，但是没有点击。 那么谷歌就会认为，你的网页吸引力不足，后面就不给你曝光了，让你回炉重造。 注意，这不是意味着你的网站被惩罚了。

这也是为什么会有今天的哥飞小课堂的原因，不少朋友问我，是不是网站被惩罚了？

哥飞很明确的告诉你，这不叫做被惩罚，这只是谷歌给你的一个信号，让你去修改网页的信号。 搞清楚了这些问题，就可以很好的指导我们，如何利用好谷歌的排名机制，做好我们的网站，我简单列一下标题，更详细的其实大家可以在 new.web.cafe 里找到，这些都是我们之前讲过的内容。

1、做好关键词调研工作 不是拍脑袋想需求，而是基于关键词和关键词的搜索量来指导我们筛选需求，确定网站主题，确定网站功能；

2、做好用的工具 做出用户体验良好的工具站，而不是粗制滥造，胡乱糊一个页面。 注重用户体验，访客进来，先给试用，之后引导登录注册，之后引导付费。而不是一上来就要求登录注册之后才能使用；

3、做精品工具页面 这个页面不仅仅要有工具，还需要有图文介绍内容，还需要有工具结果展示。 这些都有助于谷歌等搜索引擎爬虫知道我们的页面到底是干啥的，到底提供什么功能；

4、注意TDH，注意关键词密度 TDH帮助爬虫更好的识别网页的主要内容，关键词密度帮助爬虫更好的提取网页的核心主题。 3和4，其实都是为了让我们的网站能够通过正排索引和倒排索引进入到相关关键词的每一次搜索待排序列表里。

5、做好Icon、Canonical等其它小细节 很多细节工作，都需要在上线之前就做好，否则等到抓取之后，再想改，就会麻烦一些。

6、网站刚上线就要开始做外链建设工作 别听那些没有实践经验的“SEO专家”说新站不需要做外链，我们的实践经验是，上线第一天，就可以开始做外链工作了。 尤其是在爬虫来得特被频繁的网站里留下我们新站的外链，还有助于新站更快被爬虫抓取收录。 接下来就是上站之后的工作了，我们之前小课堂分享过了，可以在这里看到 [https://new.web.cafe/tutorial/detail/sk5je1bj0f](https://new.web.cafe/tutorial/detail/sk5je1bj0f)
