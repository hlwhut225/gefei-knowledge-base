# 上站之后要做什么；再次详解谷歌搜索排名；以月收入$50k的网站为例说明

今天的#哥飞小课堂 ，我们聊聊如何基于数据驱动，做好上站之后的一些宣传推广工作。

很多朋友，刚开始做的几个网站，做的事情兴冲冲，做完就发现，怎么没流量呀？

我今天就来告诉大家怎么排查问题，怎么基于数据驱动做事。

说是上站之后，是因为我默认大家做的网站，已经是做好了关键词研究工作，挖掘出来了有搜索量的关键词，并且围绕着关键词对应的需求做好了网站功能，并且每一个页面都是围绕着核心关键词去做的。

如果以上前提都没做好，你就需要去找找之前的小课堂内容去学习了。

在做好了以上的前提下，我们再来学习今天的小课堂。

既然我们选好了关键词了，那么我们做这个网站的目的，肯定是想拿到这个关键词以及相关关键词的搜索排名，有了排名才有曝光，才有点击，才有访问量。

所以我们上站后的一切动作，都是为了能够拿下排名。

做好了以上前提之后，上站第一步，提交网站到GSC（Google Search Console），等待谷歌收录我们的网站。

一般如果你的页面内容写得没问题的话，24小时内就能够被收录。

如果你想加速收录，可以把网站提交到一些谷歌爬虫来得特别频繁的网站里，这样就能够更快速的被谷歌爬虫发现你的网站，从而更快收录。

在合适的外链刺激下，最快可以2小时被谷歌收录。

如果你的内容是围绕着关键词去写的，并且核心关键词密度合理，那么网页被收录后24小时到48小时内，可以在GSC后台的效果页面看到曝光了。

也就是我们说的出词。 能够看到有哪些关键词能够拿到曝光，排名是多少。

如果到了48小时，GSC还是全无动静，那么说明你的文案写得不行，可能是内容长度不够，可能是关键词分布不合理，可能是TDH里没有覆盖关键词，可能是核心关键词密度不够。 甚至有可能你的网页内容是前端渲染的，谷歌爬虫没有抓取到内容。

以上说的这些，你都需要去一个一个细节去排查。 确保在GSC后台能够出词。

再次提醒，我们的页面，需要围绕着有搜索量的关键词去做。 如果你都没有做到这一步，那么不出词，也是有可能的。

出了词，我们还得去看，是我们想要的关键词吗？ 即使不是跟我们的核心关键词一模一样的，也需要是相关的关键词。

如果出的词都不是你想要的，说明你的文案写得有问题。 这里特别提醒，UGC内容不要误入落地页，防止稀释你的核心关键词密度。

只要出词了，就说明你前面的工作都做对了，接下来就是提升我们的网页在这些关键词下的排名。

一开始可能给你显示排名在八九十名甚至一百多名，但没关系，这时候只要你持续的去加外链，排名就能够上升。

怎么加外链？ 最好的办法就是抄作业。

通过在谷歌搜索相关关键词，找到一些竞品网站，使用 Ahrefs 或者 Semrush 导出竞品网站的外链。

你多找几个竞品网站，对他们的外链取并集，然后越多网站有的外链，你就可以越先去添加。

因为那么多网站能够拿到外链，说明这种外链比较容易获取。

尽管有别人可能跟你说，外链要注重质量而不是数量，但是你一个新网站，有什么资格挑挑拣拣，先尽快拿下第一批外链，才是你最重要去做的事情。

对于你一个全新上线，权重为0的网站，质量再差的外链，也比没有外链好，因为有了外链，就是零的突破。

对于那些竞争不激烈的关键词，尤其是新词，竞争的页面不多时，你添加几个外链之后，效果就立竿见影了，可以在GSC效果里看到排名在不断上升。

也就是说，评判一个外链对你的网站是否有用，要看你当前权重。

越到后面，你的网站权重越高，要想提升DR，所需要的外链质量就越高。

为了更好的说明后面的步骤，我再解释一下出词。

我直接把上面这篇之前写的文章内容给复制过来。

谷歌是如何抓取我们的网页，并且如何理解网页的。

注意，下文哥飞会在不同的句子里用“网址”、“网页”、“链接”等不同描述，有时候是同一个东西。

一、爬取

不管是什么网页，文章也好、新闻也好、图片页面、视频页面、音乐页面，谷歌理解网页，依然靠的是从网页获取到的文字信息。

那么是怎么获取的呢？

用的是谷歌的爬虫。

爬虫是什么？

爬虫是形象化的说法，其实是一段运行在谷歌服务器里的程序代码。

拿到一个网址后，程序发出一个 GET 请求，获取网址对应的 html 代码，这就是爬虫最主要做的事情。

注意，这里只是获取到了 html 代码，并没有去执行 html 代码。

我们可以想到，如果你的网页是前端渲染的，那么拿到的就只是 html 模板，没有任何内容，那自然就无法参与排名。

二、解析

爬到 html 代码之后，还需要用程序去解析 html 代码。

解析 html 代码目的是提取网页的正文部分，当然页头、页尾、Meta信息等也会被识别和提取出来。

以及最重要的，当前 html 页面里的所有的链接都会被提取出来。

三、索引

之后，当前页面解析出来的信息就会入库，然后进行分词、抽取主题、语义理解、建立正向索引、反向索引等步骤，这里哥飞就不展开讲解了。

（中间步骤略过）

九、关键词召回

当用户搜索一个关键词时，谷歌会找出所有内容匹配的网页。

怎么找？

根据反向索引来找。

这里哥飞就不深入讲解了，如果大家想了解，可以谷歌搜索了解更多关于搜索引擎的原理。

但是我们知道，要想网站被召回，就一定要内容足够匹配。

这就要求第一是后端渲染，能够让谷歌爬虫可以抓取到内容。

第二则要求我们即使是工具站，也要做内容，让谷歌能够根据内容知道我们的工具到底有什么作用。

十、搜索排名

被召回的结果有几千万个，怎么给网页排序呢？

谷歌最初使用 Page Rank 算法，根据不同网站的权重来进行排序。

但是经过二十多年的发展，排序算法已经复杂无比了，会根据各种因素去综合排序。

总结一下，能出词，说明在用户搜索某个关键词词，你的网页被从几千亿个页面里找出来了，参与了排名，并且进入了前200名。

而我们想要让我们的网页排名上升，其实就是想从200名不断上升到前10名。

也就是说，我们一个网页参与一个关键词排名时，不需要跟整个互联网的所有网页去比赛。

而只需要跟两百个网页去比赛。

不同的关键词，竞争程度不一样，说的就是这200个网页的实力强弱。

注意，上面这一点很重要，我希望每一个朋友都能理解。

大搜索量或者高价值的关键词，已经上线的网页们拼的你死我活，他们会调动他们能够调动的一切力量去提升网页的权重，从而拿到更靠前的排名。

你选中的关键词，好不好做，其实就是这200个页面强不强。

而只要有了一些外链，一般都可以轻松的排名进前100吧。

想要排名继续上升，就需要加更多的外链。

为什么新词好做？

因为一开始参与排名的200个网页，大多数都是弱鸡。

所以你只要做得早，甚至可以没有外链就拿下排名。

如果一直没有竞争对手出现，你的先发优势就能够一直保持。

一旦有了竞争对手了，并且他们做得比你好，你就会被他们甩下，他们的排名上升，把你就挤下去了。

这个网站，sprunkin.com 就是后来居上的典范，他的域名是10月17号买的，已经晚了20天了，但是他看准了这是一个有大搜索量的关键词，依然参与进来了。 这两三个月的时间，他的流量从3万多涨到了127万，又在12月涨到了304万。 而最近28天，他的访问量是365万，也就是说，1月份他的流量还在增长。

怎么做到的？ 他做了几件事情： 1、不断加外链，现在有878个域名给他外链了； 2、不断上游戏加内页，现在这个站被收录了1330个页面了； 3、不断收集游戏，自己放到CDN里，允许别人链接，但会在iframe里显示他的域名，又能够获得很多点击。

从他的这些动作，我看到了： 1、认准了一个能够赚钱的方向，并且竞争对手都是弱鸡三两只的情况下，用调用自己的资源，不断做大做强。 2、在大竞争对手还没入场时，持续不断的强大自身。 3、之前的第一名 Sprunki.org 外链已经停滞好久了，流量上升曲线也被 Sprunkin.com 打断了，曲线并平缓了。

我根据我自己同类游戏站的Adsense收入，推算了一下，这两个网站在12月的收入，不一定完全正确，仅供参考。

Sprunkin.com $40K 左右 Sprunki.org $120K 左右

因为每一个账号的单价都不一样，也许他们不一定有这么高，但也八九不离十了。 即使打个6折，也还有很多。

利益动人心，因为这里的需求足够大，所以他们可以调动自己的资源进去，做好各种事情。

这里我也反思了一下，我属于看到了这里的机会，但是没有投入更多资源进去，说明重视度还是不够。

说完了这个案例，我们再回到今天的主题，我们上站之后需要做什么？

其实总结起来，就两件事： 1、加精品页面（注意不会批量生成垃圾页面）； 2、加外链（越多越好，但不要去买垃圾外链）。

当我们的网页拿到了排名，有了曝光，有了点击之后，就需要密切关注流量统计平台里的一些用户行为数据指标了，主要看这三个： 1、跳出率，要低； 2、停留时间，要高，至少要比竞争对手高； 3、人均访问页面次数，要多。

如果谷歌给你流量了，但是你的网页没办法满足用户需求，那么就会导致谷歌认为你不值得，就不会给你给靠前的排名，不会给你更多的曝光，不会给你更多的点击了。

我跟很多已经走在前面的群里朋友们交流，一致认为，只要不是出现了十几年的需求，而是最近一两年因为AI的出现而出现的新的需求新的关键词，其实即使KD六七十，也是可以去做的。

不要被高KD吓倒，只要搜索量足够大，就可以去参与进来分一杯羹。
