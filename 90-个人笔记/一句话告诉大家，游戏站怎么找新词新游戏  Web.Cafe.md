# 一句话告诉大家，游戏站怎么找新词新游戏

今天的#哥飞小课堂 很短，但是很有用。

一句话告诉大家，游戏站怎么找新词新游戏。

很简单，你先找出一批做得好流量大的游戏站，这些站有个特点，都会时不时上线新的游戏子页面。

你去跟踪这些站的 siemap ，针对每个新上线的页面提取关键词，去谷歌趋势判断是否是新词，如果是，那就注册个域名上线一个网站去竞争。

游戏站单纯靠谷歌趋势很难找出新词，但是跟踪这些做得好的游戏站，你就能及时发现新词。
