# 哥飞实操演示，一次完整的找到新词的过程

很多时候，词不用刻意的去找，只要你掌握基础方法了，那么在每天的上网过程中，就能找到一些关键词。

今晚，哥飞本来是想分享某个网站，结果后来发现了一个新词。所以在这个帖子记录一下完整过程，后面大家也可以跟着这个流程走，多多练习，你也能学会找词。

这里排名第2的网站 tryhardguides.com ，月访问量272万，也可以研究一下。 现在 aitdk.com 插件可以直接在谷歌搜索结果页面显示网站访问量和域名注册时间，真的很方便了。

所以找关键词，有技巧吗？ 有，用谷歌趋势，基于词根去找就行。 方法已经告诉大家了，去执行就好了。

词根怎么来的？ 多看一些工具网站，找到一些大家都在使用的关键词，如 maker、creator、editor、checker之类的，自己收集整理起来就好了。

1.  得到关键词，习惯性的去谷歌搜索。
2.  谷歌搜索出来的网站，习惯性的点开看访问量。
3.  看网站的流量来源关键词得到更多关键词。
4.  回到第1步，继续。 每天看到100个关键词，看他1000个网站。 慢慢你就有感觉了。

![image.png](%E5%93%A5%E9%A3%9E%E5%AE%9E%E6%93%8D%E6%BC%94%E7%A4%BA%EF%BC%8C%E4%B8%80%E6%AC%A1%E5%AE%8C%E6%95%B4%E7%9A%84%E6%89%BE%E5%88%B0%E6%96%B0%E8%AF%8D%E7%9A%84%E8%BF%87%E7%A8%8B%20%20Web.Cafe/7a91d69fd3bb4fb5b25fd03566f66e52.png)
