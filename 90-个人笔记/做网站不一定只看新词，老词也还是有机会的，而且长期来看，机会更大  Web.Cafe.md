# 做网站不一定只看新词，老词也还是有机会的，而且长期来看，机会更大

对比去年12月的各个关键词能够获取的流量，再看今年10月的各个关键词能够获取的流量变化，明显比之前少了很多。

如果有人告诉你，更换域名，只要做好301，就不会对排名有影响，那么对方一定是骗子，或者没有实际实践经验。

有一种情况，不太会变化，那就是你的品牌词是独一无二的，没人跟你抢时，你的新站和老站，都有相同的品牌词，那么不会有太大的损失。 但一旦有人跟你抢某个词的排名，你换域名了，就不太能保住位置了。

这两个词，本来是那个老站长的，现在已经被群友@奥赛 的网站抢了。 所以，做网站，你一旦犯错，就会被人趁虚而入，你想要再抢回之前的位置，就没有那么容易了。

而群友的网站为什么能够抢走对方的位置？ 因为已经提前几个月做好了“准备”工作。

如同他不是在之前听了小课堂的分享后就去做了这样的网站，也不可能等到几个月之后的机会。

当然我们不是说，一定要等什么机会，因为这样的机会出现概率很小。

但网络世界，变数太多，我们做好我们的，就足够拿到一些访问量了。

我们群里很多人的案例都说明了，老词，花几个月的时间去做，也是可以拿到排名和访问量的。

给大家看另一个老词的竞争对手占有份额变化曲线，红色箭头指向的那个网站，是群友做的，硬生生从零开始，挖下来了一块不错的蛋糕。 ![image.png](%E5%81%9A%E7%BD%91%E7%AB%99%E4%B8%8D%E4%B8%80%E5%AE%9A%E5%8F%AA%E7%9C%8B%E6%96%B0%E8%AF%8D%EF%BC%8C%E8%80%81%E8%AF%8D%E4%B9%9F%E8%BF%98%E6%98%AF%E6%9C%89%E6%9C%BA%E4%BC%9A%E7%9A%84%EF%BC%8C%E8%80%8C%E4%B8%94%E9%95%BF%E6%9C%9F%E6%9D%A5%E7%9C%8B%EF%BC%8C%E6%9C%BA%E4%BC%9A%E6%9B%B4%E5%A4%A7%20%20Web.Cafe/28dd2c04facd4241ae8a3805b7687f33.png) 如果整体蛋糕没有变大的话，你抢多了，一定是别人少了。 而别人少了，并不一定是对方做了什么错误的操作（举例并没有换域名），可能只是因为你变强了，顺便就把对方给比下去了。 这就是小排老师说的软柿子理论，找到那些可以捏的软柿子，捏爆他。

再给大家看一个老词，这个词的蛋糕是在不断变大的。 水涨船高，即使你的排名没有太大的变化，流量也能增多。 ![image.png](%E5%81%9A%E7%BD%91%E7%AB%99%E4%B8%8D%E4%B8%80%E5%AE%9A%E5%8F%AA%E7%9C%8B%E6%96%B0%E8%AF%8D%EF%BC%8C%E8%80%81%E8%AF%8D%E4%B9%9F%E8%BF%98%E6%98%AF%E6%9C%89%E6%9C%BA%E4%BC%9A%E7%9A%84%EF%BC%8C%E8%80%8C%E4%B8%94%E9%95%BF%E6%9C%9F%E6%9D%A5%E7%9C%8B%EF%BC%8C%E6%9C%BA%E4%BC%9A%E6%9B%B4%E5%A4%A7%20%20Web.Cafe/1680886ff8a94e78876b9b8a34e7074a.png)
