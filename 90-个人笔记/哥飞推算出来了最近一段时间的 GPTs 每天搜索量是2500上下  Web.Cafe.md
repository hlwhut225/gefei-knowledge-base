# 哥飞推算出来了最近一段时间的 GPTs 每天搜索量是2500上下

之前，我们通过一些方式，推算出来 GPTs 的每天搜索量在 5K 左右。

但如果我们现在去看谷歌趋势，GPTs 的最近30天热度趋势的话，会发现曲线在向下走，也就是说，热度其实在下降。

![image.png](%E5%93%A5%E9%A3%9E%E6%8E%A8%E7%AE%97%E5%87%BA%E6%9D%A5%E4%BA%86%E6%9C%80%E8%BF%91%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E7%9A%84%20GPTs%20%E6%AF%8F%E5%A4%A9%E6%90%9C%E7%B4%A2%E9%87%8F%E6%98%AF2500%E4%B8%8A%E4%B8%8B%20%20Web.Cafe/f10fa6bf94a14224bc375c34b604f96f.png)

那要怎么推断出来，GPTs 的真实搜索量是多少呢？

已知，在Ads的关键字规划工具里，是无法查询 GPTs 的搜索量的，因为系统把 GPTs 当作 GPT ，直接合并了搜索量，如下图所示，虽然我输入时同时写了 GPTs 和 GPT ，但是最终出来的只有一个词的数据，因为被合并了。

![image.png](%E5%93%A5%E9%A3%9E%E6%8E%A8%E7%AE%97%E5%87%BA%E6%9D%A5%E4%BA%86%E6%9C%80%E8%BF%91%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E7%9A%84%20GPTs%20%E6%AF%8F%E5%A4%A9%E6%90%9C%E7%B4%A2%E9%87%8F%E6%98%AF2500%E4%B8%8A%E4%B8%8B%20%20Web.Cafe/6b2576a062704621b320245f78566a6e.png)

那么，最好的办法，就是基于一个已知搜索量的关键词去推算了。

刚好哥飞最近有个站，拿到了首页的排名，在 GSC 查看这个词的曝光量，就可以简单当作这个词的每天搜索量。

![image.png](%E5%93%A5%E9%A3%9E%E6%8E%A8%E7%AE%97%E5%87%BA%E6%9D%A5%E4%BA%86%E6%9C%80%E8%BF%91%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E7%9A%84%20GPTs%20%E6%AF%8F%E5%A4%A9%E6%90%9C%E7%B4%A2%E9%87%8F%E6%98%AF2500%E4%B8%8A%E4%B8%8B%20%20Web.Cafe/0fe7d358af384faca309edc2c68c1288.png)

不过从上图可以看到，有几天的曝光量下降了，这到底是排名下降导致曝光量下降了，还是真的搜索量下降了呢？

我们就需要去谷歌趋势里导出数据，来基于热度变化，修复一下搜索量。

下图红色曲线是 GPTs ，蓝色和黄色分别是已知关键词和带双引号关键词的热度变化曲线。

![image.png](%E5%93%A5%E9%A3%9E%E6%8E%A8%E7%AE%97%E5%87%BA%E6%9D%A5%E4%BA%86%E6%9C%80%E8%BF%91%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E7%9A%84%20GPTs%20%E6%AF%8F%E5%A4%A9%E6%90%9C%E7%B4%A2%E9%87%8F%E6%98%AF2500%E4%B8%8A%E4%B8%8B%20%20Web.Cafe/e59ae9491db2438784cc50bd6ebee0f5.png)

可以看到，加不加双引号，热度很接近，这就说明这个词，大家都是这样搜索的，没有其它说法。

把谷歌趋势里的数据导出，再跟 GSC 里的数据一起发给 Claude ，让他写代码去修复数据。

![image.png](%E5%93%A5%E9%A3%9E%E6%8E%A8%E7%AE%97%E5%87%BA%E6%9D%A5%E4%BA%86%E6%9C%80%E8%BF%91%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E7%9A%84%20GPTs%20%E6%AF%8F%E5%A4%A9%E6%90%9C%E7%B4%A2%E9%87%8F%E6%98%AF2500%E4%B8%8A%E4%B8%8B%20%20Web.Cafe/52ec18636724426c96c4eae0c4cd3b41.png)

从上图可以看到，12月17和18两天的搜索量被修复了，但即使修复之后，曲线也是下降的，说明那几天，的确整体的搜索量就下降了。

接着，再把 GPTs 的谷歌趋势相对热度放进去，让 Claude 去推算 GPTs 的搜索量，最终得到下面的这张图片。

![image.png](%E5%93%A5%E9%A3%9E%E6%8E%A8%E7%AE%97%E5%87%BA%E6%9D%A5%E4%BA%86%E6%9C%80%E8%BF%91%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E7%9A%84%20GPTs%20%E6%AF%8F%E5%A4%A9%E6%90%9C%E7%B4%A2%E9%87%8F%E6%98%AF2500%E4%B8%8A%E4%B8%8B%20%20Web.Cafe/d19916c394f44eeb87cad118e261e9bf.png)

可以看到，从12月15日开始，GPTs 的搜索量已经变成 2500 上下了。 而12月15日之前，的确是 5000 上下。
