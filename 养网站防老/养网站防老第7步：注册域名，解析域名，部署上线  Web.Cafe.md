# 养网站防老第7步：注册域名，解析域名，部署上线

大家好，我是哥飞。

养网站防老我们终于写到第7步了，先回顾一下前面的步骤：

在前面步骤中，我们把静态网页做好了，大家会发现，手动去制作这么多页面，再给每个页面做三种语言，工作量很大。

其实我们会发现，很多都是重复的工具，只需要做好一个国家网页模板，去批量替换就可以生成其他所有国家的页面，只需要做好一种语言，再抽取语言包出来，就可以生成其它语言的页面。

这就需要用到脚本生成页面了，不管你用js还是php，还是python或者java，还是go语言，任何语言都可以用，你会哪种语言就用哪种。

你说，你只会js连nodejs都不会怎么办？那就用js渲染生成好代码，然后手动复制浏览器里的html代码也行。

总之，学会变通，有很多办法可以提升效果。

说到这里，哥飞要跟大家检讨一下，哥飞虽然“说得好听”，上面的方法跟大家说了，但是因为个人时间原因，并没有把手机号生成网站的所有页面都做完，才导致推文时间一拖再拖，哥飞今天不拖了，先上线第一版再说吧。

大家也可能会遇到类似的情况，记住一点，有比没有好，完成比完美更重要，请尽快上线第一版。

一、注册域名

通常来讲，我们做这类关键词搞流量的网站，那么最好直接拿关键词去注册域名，我们先用 phonenumbergenerator.com 试一下，结果发现域名已经被人注册了。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/b05b150b8f56434ca2449ba787351ab7.png) 那就再用 phone-number-generator.com 去测试，好像也被注册了？ ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/2198f98cbc524e9daa999ee9ef9c8cab.png) 别担心，是哥飞在写第一步时提前注册好了的，这样到了今天就不会出现无域名可用的情况。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/bdd3a2e59a5f49b391493b5397fd3ba5.png) 一般的原则是先看关键词com是否能用，不能用就看加横杠的com，如果还不能用，就看关键词net，再看加横杠的net。如果com和net都没有了，再去考虑用其它域名后缀。

三、提交代码到 Github 仓库

打开 Github ，在右上角点击“+”号，选择新建一个仓库。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/e4f165302ee6442eb2ae86f9790bcff2.png) 可以直接用域名当作仓库名称，描述随便写都行，选择“Private”私有仓库，然后点击右下角创建仓库按钮即可。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/8e557eeb21d44feabdaf626d54102fe9.png) 之后界面会有提示，要你在自己电脑里创建一个本地仓库，把代码放进去，然后提交。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/c368f6888ec74ca2953a0efdd02d5ca4.png) 本地如何提交代码，大家可以找找git教程看看，哥飞本文不展开。提交成功后，刷新仓库，可以看到的界面如下，所有提交上来的代码文件都能看到。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/f7d4135b533a48b1899a2fcd4755eb80.png) 四、在 Vercel 部署

Github 仓库中有代码后，就可以部署到Vercel 中了。

需要提醒一下，如果你用全新注册的 Github 账号去注册 Vercel ，那么很有可能刚注册号就被封号。这时候你可以邮件联系客服，证明你是真人，而不是薅羊毛的机器人，就能解封账号了。

打开 Vercel 控制台，点击网站列表右上角的“Add New”按钮，选择新建一个“Project”。 我们这种纯静态页面，不需要做什么配置，直接在下一个页面点击“Deploy”按钮部署即可。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/f0318021b3c14500b0af984b10f041c0.png) 部署成功后，可以看到撒花界面。点击右上角的“Continue to Dashboard”按钮。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/03e63b4842cf47dea54076ca9ebc73fb.png) Vercel 送了一个子域名给我们测试用，可以点击这个域名看看网页，但我们不用这个子域名，我们点击右上角的“Domains”按钮，进入自定义域名配置界面。 在输入框输入我们的域名，然后点击“Add”按钮。 打开 Cloudflare 的网站列表，点击域名DNS解析记录，点击“Add record”按钮。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/f2db855f22f04108a2a195d8e15f14f1.png) 按照 Vercel 的要求，www域名添加 CNAME 记录，按照下方截图所示填写好，点击保存。 再用同样的方法，配置不带www域名的A记录解析。 此时，如果你用浏览器打开域名，会发现报错了。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/b9234f9de2954ca497deed011d251a7a.png) 这是因为我们还有一个步骤忘了操作，打开 Cloudflare ，进入 SSL/TLS 界面，会发现目前的模式是“Flexible”，我们需要调整为“Full”模式。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/34079b4211904189b6b3af8859fde51b.png) 看到提示“mode is Full”，就可以了。 ![image.png](%E5%85%BB%E7%BD%91%E7%AB%99%E9%98%B2%E8%80%81%E7%AC%AC7%E6%AD%A5%EF%BC%9A%E6%B3%A8%E5%86%8C%E5%9F%9F%E5%90%8D%EF%BC%8C%E8%A7%A3%E6%9E%90%E5%9F%9F%E5%90%8D%EF%BC%8C%E9%83%A8%E7%BD%B2%E4%B8%8A%E7%BA%BF%20%20Web.Cafe/7ec4280feb9b43baa4d8015d96ac8add.png) 好好欣赏你的第一个网站吧。
